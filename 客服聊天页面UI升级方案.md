# 客服聊天页面UI升级方案

## 📊 设计对比

### 当前设计（Telegram风格）
```
┌──────────────────────────────────────┐
│  [返回]   [专属客服]   [头像]         │
├──────────────────────────────────────┤
│  余额: $500  待处理: $100  客户: 5    │
├──────────────────────────────────────┤
│  [客户1] [客户2] [客户3] → 横向滚动   │
├──────────────────────────────────────┤
│                                      │
│         聊天消息区域                  │
│         (绿色渐变背景)                │
│                                      │
└──────────────────────────────────────┘
```

**问题：**
- ❌ 客户列表横向滚动，不方便查看
- ❌ 客户多了需要左右滑动才能找到
- ❌ 无法快速搜索客户
- ❌ 背景绿色渐变不够商务

---

### 新设计（微信/QQ风格）✨
```
┌─────────────┬──────────────────────────┐
│   🔍搜索    │  [客户1头像]  客户1        │
├─────────────┤  ────────────────────    │
│ 👤 客户1    │  你好，我想卖卡            │
│    [图片]   │  [图片]                   │
│    12:30 ●1 │  好的，请问...             │
├─────────────┤                          │
│ 👤 客户2    │  ────────────────────    │
│    订单查询  │  [输入框]        [发送]   │
│    昨天      │                          │
├─────────────┼──────────────────────────┤
│ 👤 客户3    │  客户: 5    余额: $500    │
│    提现申请  │                          │
│    周一      │                          │
└─────────────┴──────────────────────────┘
  客户列表       聊天区域        统计栏
```

**优势：**
- ✅ 左侧完整客户列表，一目了然
- ✅ 支持搜索客户，快速定位
- ✅ 每个客户显示最后消息和时间
- ✅ 未读消息红点提醒
- ✅ 在线状态显示（绿点）
- ✅ 响应式设计，手机端自动切换

---

## 🎨 新界面设计亮点

### 1. 左侧客户列表
- **搜索框**：实时搜索客户用户名
- **客户卡片**：
  - 头像（带在线状态绿点）
  - 用户名（粗体）
  - 最后一条消息预览
  - 消息时间
  - 未读消息红色角标
- **激活状态**：当前选中客户浅蓝色高亮
- **底部统计栏**：客户数量、总余额

### 2. 右侧聊天区域
- **顶部导航**：
  - 客户头像和名称
  - 在线状态（绿色"在线"）
  - 视频通话按钮
  - 更多操作按钮
- **消息区域**：
  - 日期分隔线
  - 对方消息靠左（白色气泡）
  - 自己消息靠右（绿色气泡）
  - 消息时间和已读状态
- **输入区域**：
  - 工具栏：相机、GIF、群发
  - 输入框
  - 发送按钮（有内容时变绿）

### 3. 粘贴图片预览
- 粘贴图片后显示预览条
- 缩略图 + 提示文案 + 关闭按钮
- 点击发送才上传

### 4. 响应式设计
- **桌面端（>768px）**：左右分栏显示
- **移动端（<768px）**：
  - 默认显示客户列表
  - 点击客户后切换到聊天页面
  - 聊天页左上角有返回按钮

## 🚀 如何使用新界面

### 方法1：替换现有页面（推荐测试后再用）

```bash
# 备份原文件
cp src/pages/support/index.vue src/pages/support/index.vue.backup

# 使用新文件
cp src/pages/support-chat-v2/index.vue src/pages/support/index.vue
```

### 方法2：添加为新路由（推荐）

**1. 在 `pages.json` 中添加新路由：**

```json
{
  "path": "pages/support-chat-v2/index",
  "style": {
    "navigationBarTitleText": "客服工作台"
  }
}
```

**2. 修改登录后跳转路由：**

在 `AuthController.java` 或前端登录逻辑中，将客服登录后的跳转地址改为：

```javascript
// 原来
nextRoute: "/pages/support/index"

// 改为
nextRoute: "/pages/support-chat-v2/index"
```

**3. 测试访问：**

```
http://127.0.0.1:5174/#/pages/support-chat-v2/index
```

## ⚙️ 技术实现细节

### 1. 响应式断点
```javascript
// 检测移动端
function checkMobile() {
  isMobile.value = window.innerWidth < 768
}
```

### 2. 搜索过滤
```javascript
const filteredConversations = computed(() => {
  const keyword = searchKeyword.value.trim().toLowerCase()
  if (!keyword) return store.state.supportConversations
  return store.state.supportConversations.filter(conv =>
    conv.customerUsername.toLowerCase().includes(keyword)
  )
})
```

### 3. 粘贴图片监听
```javascript
// 监听粘贴事件
document.addEventListener('paste', handlePasteImage)

async function handlePasteImage(event: ClipboardEvent) {
  const item = Array.from(event.clipboardData?.items || [])
    .find(entry => entry.type.startsWith('image/'))
  const file = item?.getAsFile()
  if (!file) return
  
  event.preventDefault()
  pendingImageUrl.value = URL.createObjectURL(file)
}
```

### 4. 音效播放
```javascript
// 只对用户主动消息播放提示音
function shouldPlayIncomingSound(message, conversationId) {
  return isAgent.value 
    && message.author !== 'me' 
    && message.author !== 'system'
}
```

## 🚀 部署步骤

1. **备份旧文件**
   ```bash
   cp src/pages/support/index.vue src/pages/support/index.vue.backup
   ```

2. **配置路由**（在 `pages.json` 中添加）
   ```json
   {
     "path": "pages/support-chat-v2/index",
     "style": {
       "navigationBarTitleText": "客服工作台",
       "enablePullDownRefresh": false
     }
   }
   ```

3. **修改跳转路由**（登录成功后）
   ```javascript
   // 客服登录成功后跳转
   nextRoute: "/pages/support-chat-v2/index"
   ```

4. **测试新界面**
   - 客服账号登录
   - 检查客户列表显示

   - 测试粘贴图片
   - 测试群发

5. **确认无误后替换**
   ```bash
   cp src/pages/support-chat-v2/index.vue src/pages/support/index.vue
   ```
